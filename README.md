# Linux System Monitoring Tool

A complete monitoring solution that collects CPU, RAM, disk, and network metrics from Linux systems and displays them on a dashboard.

## Features
- Agent collects system metrics
- Server API stores metrics in SQLite
- Dashboard charts real-time data
- Extendable for alerts (email, Slack)

## How to Run
### 1. Start server
### 2. Run agent
### 3. Open dashboard
Open `dashboard/index.html` in browser.
